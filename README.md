# 🤖 Thanuja.k AI Chat Assistant API

## Project Overview
AI Chat Assistant is a web-based chatbot developed using **Python**, **FastAPI**, and the **Google Gemini API**. The chatbot accepts user queries, generates intelligent AI responses, and continues the conversation until the user exits.

---

## Features
* 💬 Accepts user input
* 🤖 Generates AI responses using Gemini API
* 🔄 Supports continuous conversation
* 📝 Displays formatted responses (Swagger UI)
* ⚡ Built using FastAPI

---

## Technologies Used
* Python
* FastAPI
* Google Gemini API
* python-dotenv
* uvicorn
