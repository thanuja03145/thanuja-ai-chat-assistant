# 🤖 Thanuja.k AI Chat Assistant API

## 📝 Project Overview
AI Chat Assistant is a web-based chatbot developed using Python, FastAPI, and the Google Gemini API.

---

## 🛠️ Installation

### 1. Clone the Repository
git clone https://github.com/thanuja03145/thanuja-ai-chat-assistant

### 2. Open the Project
cd thanuja-ai-chat-assistant

### 3. Create Virtual Environment
python -m venv venv

### 4. Activate Virtual Environment
venv\Scripts\activate

### 5. Install Dependencies
pip install -r requirements.txt

### 6. Create .env File
GEMINI_API_KEY=YOUR_API_KEY

### 7. Run the Project
python app.py

---

## 📌 Required Dependencies
* FastAPI
* uvicorn
* google-generativeai
* python-dotenv
* pydantic
